## reqhfsql
Skill Claude pour les requêtes SQL compatibles avec le moteur HFSQL pour Windev

## Installation dans Claude Caude
Télécharger le zip
Extraire dans le dossier skills de votre Claude global. 
Renommer le dossier en enlevant le -main
Supprimer le fichier README.md

## Installation dans Claude Desktop
Télécharger le zip
Dans le menu Personnaliser > Compétences > Bouton + > Créer une compétence > Téléverser une compétence
Glisser le fichier zip

## Exemple d'utilisation
/reqhfsql Corrige cette requête
