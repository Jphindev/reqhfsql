# Commandes SQL disponibles dans une requête SQL

## Présentation 

WINDEV, WEBDEV et WINDEV Mobile permettent d'exécuter simplement des requêtes en code SQL. Ces requêtes peuvent être :

*   soit saisies directement sous l'éditeur de requêtes. La requête est alors exécutée avec la fonction [HExécuteRequête](https://doc.pcsoft.fr/fr-FR/?3044080&name=hexecuterequete_fonction "HExécuteRequête (Fonction)").
*   soit directement saisies dans la fonction [HExécuteRequêteSQL](https://doc.pcsoft.fr/fr-FR/?3044084&name=hexecuterequetesql_fonction "HExécuteRequêteSQL (Fonction)").
